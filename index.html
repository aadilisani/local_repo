A repository, or repo, is a centralized digital storage that developers use to make and manage changes to an application's source code. A Git repository is a virtual storage of your project. It allows you to save versions of your code, which you can access when needed. Question 4: How do you create a new Git repository? To create a new Git repository, you can either initialize a new, empty repository in a directory using git init or clone an existing repository using git clone. 1. Creating a new, empty repository (using git init): Step 1: Create a directory for your project: Navigate to the location where you want to create your project directory and create a new folder (e.g., my-project). Step 2: Open the directory in Git Bash (or your terminal): Open a terminal or Git Bash and navigate to the newly created directory. Step 3: Run git init: Type git init in the terminal and press Enter. This command initializes a new, empty Git repository in the current directory, creating a hidden .git folder. Step 4: Add files and folders to your project Step 5: Routinely stage files and create commits 2. Cloning an existing repository (using git clone): Step 1: Obtain the URL of the repository you want to clone: This could be a GitHub, GitLab, or Bitbucket repository URL. Step 2: Open a terminal or Git Bash and navigate to the desired location: Choose where you want to create the local copy of the repository. Step 3: Run git clone : Type git clone followed by the repository URL and press Enter. This command creates a local copy of the repository in a new directory with the same name as the repository.
